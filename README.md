# CDL Training Site (GitHub Pages)

## Quick start
1. Upload these files to the ROOT of your GitHub Pages repo (same folder as index.html).
2. Make sure GitHub Pages is enabled: Repo Settings → Pages → Deploy from branch → main / (root).
3. Open your GitHub Pages URL and sign in.
